# 🌌 Nakshatra - The Cosmic Destiny Engine

> *"Your Destiny Is Written In The Stars... and decoded by AI."*

Nakshatra is a full-stack, AI-powered astrology and numerology web application. By entering their birth details, users can unlock deeply personalized cosmic insights, blending ancient Vedic astrology principles with the cutting-edge reasoning capabilities of the Google Gemini API.

## ✨ Features

- **🔮 Precision Numerology:** Automatically calculates your *Mulank* (Psychic Number) and *Bhagyank* (Destiny Number) using custom mathematical algorithms.
- **🪐 Planetary Snapshot:** Estimates ascendant signs and key planetary placements across the 12 astrological houses based on your exact time and location of birth.
- **❤️ Love & Relationships:** Reveals what the cosmic alignment indicates for your romantic life.
- **💼 Career & Wealth Trajectory:** Suggests optimal career paths and financial timelines written in your stars.
- **🛡️ Remedies & Success:** Provides actionable, mystical advice and potential remedies for overcoming life's obstacles.
- **🤖 Powered by Google AI Studio:** Utilizes the `@google/genai` SDK and the `gemini-2.5-flash` model to dynamically generate warm, mystical, and highly structured astrological readings.

## 🛠️ Tech Stack

**Frontend:**
- React (Vite / CRA)
- Tailwind CSS (For glassmorphism & cosmic animations)
- Lucide React (Icons)
- React Markdown (For rendering structured API responses)

**Backend:**
- Node.js & Express.js
- Google Gen AI SDK (`@google/genai`)
- CORS & Dotenv for secure environment management

**Deployment:**
- **Frontend:** Vercel
- **Backend:** Render
